Tip Tracker Commands

Main Window Commands

FILE MENU

New - Creates a file for a new pay period. The pay period start date must be set and the pay period end date will automatically be set to include all of the days in the two-week pay period.

Open - Opens a previously created file.

Close - Closes the current file.

Save - Saves the current file.

Save As... - Saves the current file under a different file name.

Exit - Exits the program. If there are any unsaved files, you will be prompted to save them.

TOOLS MENU

Manage Template Servers - Opens the template server management window so you can add, edit, or delete template servers. See related topic About Template Servers.

Print Server List - Prints a list of all servers in the template.

Export Server List - Exports the template servers to an XML file so they can be imported to Tip Tracker SF.

Manage Users - Opens the users management window so you can add, edit, and delete users and set authorizations for each user.

Settings - Opens the settings window so you can change the default directory where files are stored.

WINDOW MENU

Cascade - Cascades all open tip entry windows.

Minimize All - Minimizes all open tip entry windows.

Maximize All - Maximizes all open tip entry windows.
