Sorting the Tip Grids

Regardless of which type of tip you are viewing, the tip grids can all be sorted in the same manner. By default, all tips are sorted by server number. To change how the tips are sorted, click the column header on the grid.

For example, if you want to sort room charge tips by last name:

  1. Select the Room Charge tab if it is not already selected.
  2. On the room charge tip grid, click the column header labeled Last Name to sort the tips from A to Z.
  3. Click the column header again to sort the tips from Z to A.

Sorting by Order Entered

Because tips are sorted by server number by default, it makes it difficult to check each entry when the tips for a particular day do not balance. However, the tips can be sorted by the order they were entered to make it easier to check each tip against the list from which it was entered.

To sort the tips by the order in which they were entered, click the Operations drop down button then click Sort By Order Entered. This will cause all of the tip grids' sort modes to be changed and a check mark will appear next to the Sort By Order Entered menu item. NOTE: While in this mode, you cannot change how the tips are sorted by clicking the tip grid column headers. To exit this mode, follow the same steps that were given to enter this mode.
