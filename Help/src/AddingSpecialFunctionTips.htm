Adding, Editing, or Deleting Special Functions Tips

Before you add a special functions tip, you must first define the function for which the tip is being paid.

Adding Special Functions Tips

To add a special functions tip:

  1. Select the Special Function tab if it is not already selected.
  2. Select the function for which the tip is being paid. When you change the selected function, only the tips for that function will be shown.
  3. Enter the server's name.
    4. NOTE: The name is searchable by first or last name.
  4. Enter the tip amount.
  5. Click Add.

Editing Special Functions Tips

In the event that a tip was added incorrectly, the tip can be edited to change the amount, working date, or type. If you want to change the server the tip belongs to, you must reassign the tip. To edit a special functions tip:

  1. Select the Special Function tab if it is not already selected.
  2. Select the tip you wish to edit.
  3. Right click then click Edit Selected.
  4. Enter the new amount, working date, and/or tip type.
  5. Click OK.
    6. NOTE: If the tip type is Special Function when you click OK, you will be prompted to select the function.

Reassigning Special Functions Tips

If you discover that a tip was entered for a particular server, and that the tip actually belongs to a different server, you may reassign the tip to the correct server. To reassign a tip:

  1. Select the Special Function tab if it is not already selected.
  2. Select the tip you wish to reassign.
  3. Right click then click Reassign Selected.
  4. Select the server who will receive the tip after reassignment.
  5. Click OK.

Deleting Special Functions Tips

Sometimes editing and reassigning are not enough and a tip just needs to be deleted. To delete a tip:

  1. Select the Special Function tab if it is not already selected.
  2. Select the tip you wish to delete.
  3. Right click then click Delete Selected.
  4. When prompted to confirm the deletion, click Yes to delete the tip or No to cancel.
