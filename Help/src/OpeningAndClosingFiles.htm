Opening Files

When Tip Tracker is installed, a default directory is set for storing data files. When opening files, the dialog box automatically starts in the default data folder. To open a file:

  1. Click the File menu then click Open... or type Ctrl + O on the keyboard.
  2. Browse to the folder where the desired file is stored.
  3. Click the desired file then click Open or double-click the desired file.

A new window will open showing the server list and the tabs for the different types of tips (with each tab containing a tip entry form and tip grid). You are ready to start tracking tips.

Closing Files

To close a file, click the File menu then click Close. You may also type Ctrl + Shift + Q on the keyboard
