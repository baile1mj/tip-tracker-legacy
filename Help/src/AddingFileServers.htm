Adding Servers

Adding servers to a data file is much like adding servers to the template. To add a server to a data file, first open the desired file, then:

  1. Right click the servers list then click Add...
  2. Enter the server number.
    3. NOTE: You cannot enter a number that has already been assigned to another server.
  3. Enter the first and last name.
  4. If you do not want a tip chit to print for this server, check the Suppress Tip Chit box.
  5. Click OK.

Tip Tracker will check to see if the server you just added exists in the template. If it does not, you will be asked if you want to add the server to the template. If you do, click Yes otherwise click No. Remember, the template is for servers who recieve tips regularly. If the server you just added does not receive tips on a regular basis, you should not add the server to the template.
