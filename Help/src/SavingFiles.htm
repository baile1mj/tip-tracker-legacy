Saving Files

To save a file, click the File menu, then click Save. You may also type Ctrl + B on the keyboard.

Saving a File to a Different Name

To save a file under a different name:

  1. Click the File menu then click Save As...
  2. Browse to folder where you want the new file to be stored.
  3. Type the new file name then click Save.
