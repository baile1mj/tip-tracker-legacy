Auto Adding Servers

In the event that there are one-time tips to be paid to multiple people, you can use the Auto Add Servers function to add those people to a data file. Auto Add starts with a server number you enter and allows you to enter names continuously until all the people are added. The first server number is used for the first name you enter, then each subsequent server number is incremented by one. To auto add servers:

  1. Click the Operations drop down button, then click Auto Add Servers...
  2. Enter the starting server number.
    3. NOTE: You may not use a server number that has already been assigned. It is recommended that you use a number that is higher than the highest number in the list of servers.
  3. Check whether you want to suppress tip chits on these servers.
    4. NOTE: Checking this option will not prevent you from changing the option for individual servers. It merely determines whether the box will be checked or unchecked by default during the Auto Add process.
  4. Enter the first and last names for the server.
  5. Click Next to continue or Done to complete the process.
  6. When prompted to save the file, click Yes to save, No not to save, or Cancel to undo addition of servers that were auto added.

NOTE: Auto Add Servers does not provide the option to add the servers to the template.
