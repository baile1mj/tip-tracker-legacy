Room Charge Tips

Room charge tips are the tips received on a restaurant check that was charged to a guest room.

Adding Room Charge Tips

To add a room charge tip:

  1. Select the Room Charge tab if it is not already selected.
  2. Enter the server number.
  3. Enter the tip amount.
  4. Click Add.

Editing Room Charge Tips

In the event that a tip was added incorrectly, the tip can be edited to change the amount, working date, or type. If you want to change the server the tip belongs to, you must reassign the tip. To edit a room charge tip:

  1. Select the Room Charge tab if it is not already selected.
  2. Select the tip you wish to edit.
  3. Right click then click Edit Selected.
  4. Enter the new amount, working date, and/or tip type.
  5. Click OK.
    6. NOTE: If the tip type is Special Function when you click OK, you will be prompted to select the function.

Reassigning Room Charge Tips

If you discover that a tip was entered for a particular server, and that the tip actually belongs to a different server, you may reassign the tip to the correct server. To reassign a tip:

  1. Select the Room Charge tab if it is not already selected.
  2. Select the tip you wish to reassign.
  3. Right click then click Reassign Selected.
  4. Select the server who will receive the tip after reassignment.
  5. Click OK.

Deleting Room Charge Tips

Sometimes editing and reassigning are not enough and a tip just needs to be deleted. To delete a tip:

  1. Select the Room Charge tab if it is not already selected.
  2. Select the tip you wish to delete.
  3. Right click then click Delete Selected.
  4. When prompted to confirm the deletion, click Yes to delete the tip or No to cancel.
