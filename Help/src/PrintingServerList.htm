Printing the Server List

The servers template can be printed as a list for quick reference to the servers in the template. The list contains the server number, first name, and the first initial of the last name for each server in the template. To print the server list:

  1. Click the Tools menu, then click Print Server List...
  2. Select which printer you want to print to.
  3. Click OK.

NOTE: Two copies of the server list will print on each page.
