Changing the Date

Changing the date changes the current working date of the data file. When the current working date of the data file is changed, the credit card and room charge tips are refiltered to show only the tips for the current working date.

To change the date:

  1. Click the Select Working Date button in the tool bar.
  2. Select the desired working date.
  3. Click OK.
