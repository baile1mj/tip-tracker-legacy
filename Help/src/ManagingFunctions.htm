Managing Special Functions

Special functions must be defined before tips can be added for those functions. Unlike credit card or room charge tips, special functions tips are filtered by function, rather than by date.

Adding Special Functions

To add a special function:

  1. Open the manage functions dialog:
    • Click the Operations drop down button then click Manage Special Functions... or
    • Click the Manage Functions button on the Special Function tab.
  2. Click Add...
  3. Enter the name of the special function.
  4. Enter the function date.
    5. NOTE: The function date must be a date within the pay period.
  5. Click OK.
  6. Click Close to close the manage functions dialog.

Editing Special Functions

To edit a special function:

  1. Open the manage functions dialog:
    • Click the Operations drop down button then click Manage Special Functions... or
    • Click the Manage Functions button on the Special Function tab.
  2. Click Edit...
  3. Make the desired changes to the function name and/or date.
    4. NOTE: The function date must be a date within the pay period.
  4. Click OK.
  5. Click Close to close the manage functions dialog.

Deleting Special Functions

To delete a special function:

  1. Open the manage functions dialog:
    • Click the Operations drop down button then click Manage Special Functions... or
    • Click the Manage Functions button on the Special Function tab.
  2. Select the function you wish to delete then click Delete.
  3. Click Yes to delete the function or No to cancel.
    4. NOTE: When you delete a special function, all the tips assigned to that function are deleted as well.
  4. Click Close to close the manage functions dialog.
