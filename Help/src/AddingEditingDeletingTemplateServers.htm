Adding Servers

To add a template server:

  1. Click the Tools menu, then click Manage Template Servers...
  2. On the manage template servers dialog, click Add.
  3. Enter the server number.
    4. NOTE: You cannot enter a number that has already been assigned to another server.
  4. Enter the first and last name.
  5. If you do not want a tip chit to print for this server, check the Suppress Tip Chit box.
  6. Click OK.

Editing Servers

To edit a template server:

  1. Click the Tools menu, then click Manage Template Servers...
  2. On the manage template servers dialog, click Edit
    3. NOTE: Once a server number has been entered, it cannot be changed. To change a server number, you must first delete the server then add a new server.
  3. Make the desired changes to the server
  4. Click OK

Deleting Servers

To delete a template server:

  1. Click the Tools menu, then click Manage Template Servers...
  2. On the manage template servers dialog, select the server you wish to delete
  3. Click Delete
  4. Click Yes to confirm the deletion or No to cancel
  5. Click OK
