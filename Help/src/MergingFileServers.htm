Merging Servers

When multiple people enter tips to a data file, servers are sometimes duplicated by mistake. This is especially true when entering tips for people who seldom receive tips and therefore are not in the template. If a server is assigned to two different numbers and has tips entered under both of those numbers, the duplicate can be merged with the original. To merge duplicate servers:

  1. In the servers list, select the server you want to remove from the file.
  2. Right click on the selected server then click Merge Duplicate.
  3. From the drop down box, select the server who will receive the tips.
  4. Click OK.

The tips will be transferred to the merge recipient and the duplicate server will be deleted from the file.
