Copying Servers from the Template

The easiest way to add new servers to a data file and the template is to add the servers to the file, then click Yes when prompted to add the servers to the template. However, it is easy to forget that servers are not automatically added to the data file at the same time they are added to the template. In the even that there are servers in the template that you wish to add to a data file, you copy servers into the file.

To do this, first open the data file that you wish to add the servers to. Next, you must right click the servers list then click Copy From Template. This function will copy only the servers who do not already exist in the data file.

NOTE: if a server number is assigned to two different people, the name assigned to that number in the data file will be changed to the name assigned to that number in the template.
