Editing Servers

To edit a server in a data file, you must first open the file. Then:

  1. Select the server you wish to edit in the servers list.
  2. Right click the selected server then click Edit Selected...
    3. NOTE: Once a server number has been entered, it cannot be changed. To change a server number, you must first delete the server then add a new server.
  3. Make the desired changes to the server.
  4. Click OK.

When you edit a server, the changes are applied to the server in the list as well as all of that server's tips.
