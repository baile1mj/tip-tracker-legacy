Creating New Files

Each data file contains the tips for one two-week pay period. When creating a new file, you must specify the start date of the pay period. The end date will automatically be calculated. When a new file is created, the servers from the template are copied into the file. To create a new file:

  1. Click the File menu.
  2. Click New...
  3. When the Create File dialog opens, select the desired pay period start date.
  4. Enter the desired file name and click Save or just click Save to accept the generated file name.

A new window will open showing the server list and the tip entry forms. You are ready to start tracking tips.
