Working with Template Servers

The servers template is the master list of servers who receive tips every pay period. The template is stored and managed separately from the server lists contained in each of the data files. When a new data file is created, all of the servers in the template are copied into the new data file. Template servers can be added, edited, or deleted at any time without affecting the data files.
