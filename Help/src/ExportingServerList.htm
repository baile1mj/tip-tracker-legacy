Exporting the Server List

The servers template can be exported as an XML file. The XML file can then be imported into Tip Tracker SF. To export the template server list:

  1. Click the Tools menu, then click Export Server List...
  2. Browse to the folder where you want the list saved.
  3. Type in the desired file name then click Save.
